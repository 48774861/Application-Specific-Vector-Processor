Each of the testbenches can be used to test one of the Verilog module files.
controller_tb.v tests the controller.v file.
datapath_tb.v tests the datapath.v file.
tb_multiplier_8bit_signed.v tests the truncated_multiplier_8bit.v file.
tb_top.v tests the top.v file.
tb_truncated_adder_8bit.v tests the truncated_adder_8bit.v file.
